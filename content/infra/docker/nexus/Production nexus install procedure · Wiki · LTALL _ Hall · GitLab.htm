Production nexus install procedure

Last edited by vincent.huang1 3 months ago
New page Page history Edit

install java

$ rpm -ivh jdk-8u162-linux-x64.rpm

install nexus

$ mkdir /opt/nexus
$ cd /opt/nexus3 && tar -xf nexus-3.1.0-04-unix.tar

$ cp /opt/nexus3/nexus-3.1.0-04/etc/nexus-default.properties /opt/nexus3/nexus-3.1.0-04/etc/nexus.properties
$ vi /opt/nexus3/nexus-3.1.0-04/etc/nexus.properties 
(change ip and port if you need, but do not use 0.0.0.0)
$ vi /opt/nexus3/nexus-3.1.0-04/bin/nexus.rc
(uncomment run_as_user=root)

$ ./opt/nexus3/nexus-3.1.0-04/bin/nexus start
$ tail -f /opt/nexus3/sonatype-work/log/nexus.log

nexus default account / password

  • admin / admin123

setting docker for mac insecure registries

docker push command

suppose you already build a image, boya/sport-api:0.0.1

$ docker tag boya/sport-api:0.0.1 10.10.1.31:8082/sport-api:0.0.1
$ docker login 10.10.1.31:8082 -u admin -p admin123
$ docker push 10.10.1.31:8082/sport-api:0.0.1

docker pull command

suppose you already build a image, boya/sport-api:0.0.1

$ docker login 10.10.1.31:8082 -u admin -p admin123
$ docker pull 10.10.1.31:8082/sport-api:0.0.1
