##### basic operation
```
i 
u
:q
:w
:e
:q!
```

##### search text forward and backward
```
/keyward and ?keyword
```

##### move to next keyword and last keyword
```
n and N
```

##### jump to document head and bottom
```
gg and G
```

##### copy, cut and paste
```
nyy, ndd and p
```

##### replace
```
:s/old/new
:10,20s/old/new/g
:%s/old/new/g
```

##### counting keyword
```
:%s/pattern//gn
```
