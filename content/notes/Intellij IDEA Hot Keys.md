* Selection
```
Column Selection Mode 			cmd + shift + 8

To select multiple words, follow these steps, press alt + shift and double click the left mouse button, press ctrl + g repeatly
To select multiple fragments of text, follow these steps, press alt + shift and drag mouse pointer
To select multiple rectangular fragments of text follow these steps, cmd + alt + shift and drag mouse pointer
To create a large rectangular selection, create a normal selection first, with the given starting and ending points, and then press cmd + shift + 8 to toggle to the column selection mode.

Extend Selection 				cmd + w
Shrink Selection 				cmd + shift + w
```
