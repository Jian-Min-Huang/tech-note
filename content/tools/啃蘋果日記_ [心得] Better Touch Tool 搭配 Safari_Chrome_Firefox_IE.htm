[心得] Better Touch Tool 搭配 Safari/Chrome/Firefox/IE

我覺得 Mac 平台上最值回票價的，大概就是筆電上那塊多點觸控版，以及可選購的藍芽無線 Magic Trackpad。

Chrome/Safari/Firefox 預設採用兩指捲動、兩指左右滑動切換前後頁，兩指縮放網頁，這些已經很方便了。

想要進一步發揮多點觸控的威力，可以採用 3rd-party 軟體，比較有名的是 BetterTouchTool，網路上已經有很多人介紹過這套軟體，這邊就不用贅述了。只說一下我的用法：



1. 先把 Mission Control 預設的三指控制改成四指 

如此一來，可以把三指手勢保留給瀏覽器使用。


2. 在 BetterTouchTool 中設定 Safari 的手勢

在 Safari 中，我喜歡的手勢是

  • 三指點連結，會自動開啟至新分頁
    • 把 Three Finger Tap 設定為 Middleclick (模擬滑鼠中鍵)
  • 三指往左右滑動，會切換至前後分頁
    • 把 Three Finger Swipe Left 設定為 Ctrl+Tab
    • 把 Three Finger Swipe Right 設定為 Shift+Ctrl+Tab
  • 三指連續點兩下，會關閉目前的分頁
    • 把 Three Finger Double-Tap 設定為 Command+W
  • 三指上下滑動還可以設定別的功能
    • 比如如果希望三指往上滑就切換到 Safari 的 Top Sites，可以把 Three-Finger Swipe Up 設定為 Command+Option+1
    • 我習慣把三指往上滑設定為移至頁尾 (Ctrl+下)，我喜歡把三指往下滑設定為移至頁首 (Ctrl+上)，



由於兩指連續點兩下在 OS X Lion 中預設是查字典功能，得在系統設定中關閉，否則兩邊設定會打架



這樣就可以用一個觸控板，模擬原本要多鍵滑鼠才能做到的上網介面了。很好用喔！

3. Firefox / Chrome


上述 Safari 的設定除了 Top Sites 是 Safari 特有的功能外，其餘按鍵都適用於 Firefox / Chrome。


4. 終端機

如果你跟我一樣很常使用終端機，也可以設定上面的三指左右滑動到終端機的左右分頁切換上 （對應到 Command+{ 及 Command+} ）




5. VMWare 裡頭的 IE 瀏覽器

這一套作法也適用於 VMWare 內的 Windows Firefox/IE 瀏覽器，但不適用於 BootCamp 開機的情況。 (因為 Apple 並未開放多點觸控裝置的 Windows driver)

設定方式跟上面類似，把應用程式從特定瀏覽器改成 VMWare，只是要注意：

a. Windows 內的快速鍵要把 Command 鍵替換成 Control 鍵，如 Mac 中關閉分頁按 Command+W，Windows 中要改成 Control+W

b. 如果想要模擬 Home/End 鍵 (移到網頁首尾的功能)，請按 Fn+左/右方向鍵。

c. 若想在 IE 中使用兩指左右滑動切到上下頁，可以指定 Two-Finger Swipe 為 Command+左右鍵或 Option+左右鍵 (取決於你的 VMware 設定中，是拿 Option 或 Command 按鍵當做 Alt 鍵)

d. 如果想要在 IE 中指定兩指縮放放大縮小字體，可以把 Pinch In/Out 指定為 Control -/+
